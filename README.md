# 💣 Minesweeper

A classic implementation of the timeless puzzle game, **Minesweeper**, built using modern **Kotlin** development practices, demonstrating clean architecture and functional game logic.

## 🚀 Features

- **Classic Gameplay:** All traditional mechanics including tile revealing, flagging, and win/loss conditions.
- **Modern Kotlin:** Built with clean, idiomatic Kotlin code leveraging modern development standards.
- **Functional Logic:** Decoupled game logic to ensure reliability and easy testing.
- **Responsive UI:** A clean interface designed for a smooth user experience.

## 🛠️ Built With

- [Kotlin](https://kotlinlang.org) - Primary programming language.
- Clean Architecture principles for separation of concerns.

## 🎮 How to Play

1.  **Objective:** Uncover all the tiles that do not contain mines.
2.  **Left Click:** Reveal a tile. If it's a mine, the game is over!
3.  **Right Click:** Place a flag on a tile you suspect contains a mine.
4.  **Numbers:** The number on a revealed tile indicates how many mines are adjacent to it.

## 🔧 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/raymondoyondi/Minesweeper.git
    ```
2.  **Open in IDE:**
    Import the project into [IntelliJ IDEA](https://www.jetbrains.com) or [Android Studio](https://developer.android.com).
3.  **Build & Run:**
    Sync Gradle and run the application.

## 📂 Project Structure

- `src/main/kotlin`: Contains the core game logic and UI implementation.
- `src/test/kotlin`: Unit tests for ensuring game rule integrity.
